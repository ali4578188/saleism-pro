# SALEISM PRO

Professional Wholesale Inventory & Sales Management App for Android.

## Features
- Dashboard with real-time stats
- Product management (categories, pricing, stock levels)
- Purchase entry & tracking
- Sales management with invoicing
- Company/supplier ledger
- Stock reports
- PDF & Excel export
- Biometric authentication
- Backup & restore

## Build

```bash
flutter pub get
flutter build apk --release
```

## Requirements
- Flutter >= 3.10.0
- Android SDK 23+
